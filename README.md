# 簡易記帳 (Vibe Coding 範例)

## 內容
- index.html — 前端主頁面
- style.css — 樣式
- app.js — 主要邏輯（localStorage）
- prompt.txt — 用於 Vibe Coding 的 prompt
- report.txt — 作業報告內容（可複製到 Word）

## 本地執行
1. 將資料夾放到任意靜態伺服器或直接開啟 index.html（在多數瀏覽器中開啟即可）。
2. 建議部署方式：
   - GitHub Pages: 建立 repo，將檔案推上 master/main，進入 Settings → Pages 設定來源為 main，選擇 root，等待 publish。
   - Replit: 建立 HTML/CSS/JS Repl，上傳檔案並按 Run。
